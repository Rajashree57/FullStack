Practically radio buttons are used in group. So that when one button is selected then the other already get unselected if it was previously selected. In a group of radio buttons, all the radio elements are connected.

2.HTML Radio Group

Radio buttons in the above example are not part of any group. A radio group is defined by the name attribute. When 2 or more radio buttons are in a group they have the same name attribute. When multiple radio buttons are given the same name then it becomes part of a group and when you select any radio button of that group it automatically deselects any other selected button in that group.
